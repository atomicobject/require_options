== Description
Simply extract named arguments from a hash.  Require or allow a set of keys.

== Install
  script/plugin git://github.com/atomicobject/require_options.git
  
== Example
  class Car
    include RequireOptions    
    def go(opts)
      gear, speed = require_options(opts, :speed, :gear) 
      # ...
    end    
  end
